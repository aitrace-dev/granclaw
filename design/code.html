GranClaw

Everything you wanted in OpenClaw...

GranClaw is a personal knowledge synthesis engine that bridges the gap between static notes and executable agents. Built upon the philosophy of [[claude-code]], it allows you to treat your markdown files as living documentation for your AI workforce.

Unlike standard productivity apps, GranClaw prioritizes your [[sleep-at-night]] by ensuring everything is [[self-hosted]] and private. No cloud, no tracking—just pure intellectual labor.

Install

# Clone the vault
git clone git@github.com:granclaw/core.git
# Install dependencies
npm install --global granclaw-cli
# Initialize agents
granclaw init --vault ~/brain
warning
Requirement

GranClaw requires Node 20+ and a local LLM runner (Ollama) or a valid Anthropic API key for agentic operations.
